很多刚用surge的人都会遇到一个问题就是怎么添加多个订阅？其实很简单，那就是UI新建一个策略组，策略组下面选引用外部链接就是订阅链接生成一个带有全部节点的策略组，然后在其他地区策略组或其他策略组选择包含另一个策略组的策略里面选你刚创建的那个机场策略组就行了，然后下面填上你想怎么筛选的正则就完事了。
